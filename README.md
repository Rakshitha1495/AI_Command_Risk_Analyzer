# AI-Powered Command Risk Analyzer

## 📌 Overview
This project is a smart assistant that analyzes terminal commands and detects potential risks.

## 🚀 Features
- Risk level detection (Low / Medium / High)
- Impact analysis
- Safer command suggestions
- AI-based explanation
- Clean and modern UI

## 🛠 Tech Stack
- Python
- Streamlit
- Rule-based system + AI hybrid logic

## ▶️ How to Run
1. Install requirements:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py
